O mnie

Nazywam się Paweł Januszkiewicz i aspiruje do miana front-end developera.

Zainteresowanie technologiami webowymi towarzyszyło mi jeszcze od czasów gimnazjum, niestety było niejako tłem dla formalnej edukacji.

Od ponad pół roku poświęcam cały swój czas na podnoszenie poziomu znajomości technologii front-endowych.

Jako kolejny, naturalny krok rozwoju szukam pierwszego stażu/pracy w komercyjnej firmie.

Aktualnie jestem skupiony na doskonaleniu umiejętności programowania w JS oraz na nauce korzystania z narzędzia automatyzacji pracy Gulp.

W swoich projektach staram się dbać o intuicyjność i estetykę interfejsu użytkownika i w miarę możliwości rozwijam swoją wiedzę nt. UI oraz UX designu.

Umiejętności

Html + Jade/Pug

  • Znajomość najpopularniejszych znaczników HTML
  • Sprawne korzystanie z preprocesora Jade/Pug

Css + Sass

  • Znajomość większości właściwości CSS oraz działania selektorów
  • Umiejętność korzystania z dokumentacji (MDN)
  • Ogólna znajomość flexboxa
  • Korzystam z preprocesora SASS

JavaScript + jQuery

  • Umiejętność wykonywania operacji na drzewie DOM (wyszukiwanie elementów drzewa, edycja styli, dodawanie/usuwanie klas CSS), reagowanie na zdarzenia zarówno w jQuery jak i w "czystym" JS

Inne
